Este repositório contém o material de apoio, exemplos de código e projetos desenvolvidos durante as aulas de **Tecnologia e Desenvolvimento Web** com a turma **202** da Escola Elfrida Cristino Silva no ano de 2026.

---

## 📚 Conteúdo das Aulas
O cronograma de estudos foca no desenvolvimento de competências técnicas essenciais para o mercado de tecnologia:
- **Lógica de Programação**: Fundamentos e resolução de problemas.
- **Frontend Moderno**: Estruturação com HTML5, estilização com CSS3 e interatividade com JavaScript.
- **Introdução ao Backend**: Noções básicas de servidores e fluxos de dados.
- **Cidadania Digital**: Uso ético e seguro das tecnologias.

## 🛠 Ferramentas Utilizadas
- **VS Code**: Editor de código principal.
- **Git/GitHub**: Versionamento de projetos e colaboração.
- **Navegadores**: Chrome/Firefox para testes e depuração.

## 🚀 Como usar este material
1. Navegue pelas pastas para encontrar o conteúdo referente a cada aula ou projeto.
2. Você pode baixar os arquivos individualmente ou clonar o repositório completo para o seu computador.
3. Utilize os exemplos como base para os seus próprios desafios e tarefas.
